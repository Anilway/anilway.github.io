  • if-else语句

2、do-while语句

do-while语句是一种后测试循环语句，先执行后检测是否符合条件。

常用于循环体中代码至少执行一次的情形。

初始条件;
do{
  执行代码;
  循环变化条件;
}while(循环条件)

do-while语句使用示例：

var i=0;
do{
  i+=2;
}while(i<10);
alert(i);//10

3、while语句

while语句是前测试循环语句，在循环体代码被执行前，会对条件进行求值判断。所以，循环体中的代码可能永远不会被执行。

初始条件;
while(循环条件){
    执行代码;
      循环变化条件;
      }

while语句使用示例：

var i=0;
while(i<10){
i+=2;
}

在这个例子中，只要i<10,循环就会继续。

4、for语句

  • break：跳过循环，执行结束。
  • continue：跳过本次循环，继续执行后面的循环。

for循环像是综合了前面的循环语句，它是这样写的：

for(初始值;循环条件;循环执行一次后的变化条件){
    执行代码;
}

for语句也是一种前测试循环语句，但是它具有在执行循环之前初始化变量和定义循环后要执行的代码的能力。

示例：

var count=10;
for(var i=0;i<count;i++){
    alert(i);
}

初始条件可以不使用var声明。

ECMAScript不存在块级作用域，所以i在循环体之外也可以访问。

5、for-in语句

常用于枚举对象的属性。

比如：window对象

for(var propName in window){
console.log(propName);//打印输出window对象的所有属性名称
}

例子中的propName参数可以随便命名，它代表对象的属性名，你可以用i表示，随便。

比如：

var obj={'name':'Anna','age':26,'like':'dog'};
for(var i in obj){
console.log(obj[i]);//打印输出obj对象的所有属性值
}

ECMAScript对象的属性没有顺序。因此，通过for-in循环枚举出的属性名的顺序是不可预测的。不过，所有的属性都会被枚举一次，但是返回的先后顺序因浏览器而异。

要注意的一点是：如果对象的变量值为null或undefined，for-in语句会出错。在ES5中更正了这一点，对于这种情况不会抛错，而是不执行循环体。所以，为了保证最大限度的兼容性，在使用for-in循环之前，先确认该对象的值是不是null或undefined。

8、switch语句

如果要判断3种以内，用if判断；如果超过3种，建议用switch判断。

switch(表达式){  //注意里面写的是表达式，不是条件
    case1:
        执行代码;
        break;
    case2:
        执行代码;
        break;
    case3:
        执行代码;
        break;
  default:
        执行代码;//上述条件都不成立时，执行，类似于else
}

例如：

switch(i){
  case 25:
    alert("25");
    break;
  case 35:
    alert("35");
    break;
  case 45:
    alert("45");
    break;
  default:
    alert("other");
    break;
}

等同于if语句中：

if(i==25){
  alert("25");
}else if(i==35){
  alert("35");
}else if(i==45){
  alert("45");    
}else{
  alert("other");
}

switch中可以合并几种情形：

switch(i){
    /*合并两种情形*/
  case 25:
  case 35:
    alert("25 or 35");
    break;
  case 45:
    alert("45");
    break;
  default:
    alert("other");
    break;
}

switch语句的特色：

  • 可以在switch语句中使用任何数据类型，字符串、对象都没问题
  • 每个case不一定是常量，可以是变量，甚至是表达式。
switch("hello world"){
  case "hello"+"world":    //对字符串拼接操作求值表达式。字符串拼接后和switch的参数相等。
    alert("Greeting was found.");
    break;
  case "goodbey":
    alert("Closing was found.");
  default:
    alert("Unexpected message was found.");
}

使用表达式作为case值还可以实现下面的操作：

var num=25;
switch(true){
  case num<0:
    alert("小于0");
    break;
  case num>=0&&num<=10:
    alert("在0和10之间");
    break;
  default:
    alert("超过10");
}

之所以给switch传递表达式true，是因为每个case值都可以返回一个布尔值。这样，每个case按照顺序被求值，直到找到匹配的值或者遇到default语句为止。

9、函数

  • return：如果函数没有写return，或者不带返回值，函数停止执行后将返回undefined。
  • 不能把函数或参数命名为eval或arguments，参数之间不能重名。
  • JS函数没有重载，如果有两个同名函数，那么后一个函数会覆盖前一个函数。

重载，简单说，就是函数或者方法有相同的名称，但是参数列表不相同的情形，这样的同名不同参数的函数或者方法之间，互相称之为重载函数或者方法。

参数

  • 函数的参数可以有任意个，可以是任意类型。参数以数组的形式保存，可通过arguments对象访问。
  • arguments很像数组，但是只有length属性，没有数组其他属性和方法。
    • arguments的值永远与对应参数的值保持同步
    • 严格模式下，不能通过arguments[i]=value为参数赋值
  • 没有赋值的参数将被自动命名为undefined。

其他

函数还有很多可以研究的，比如：

  • 函数的参数可以是函数（这个作为参数的函数就被称为回调函数）
  • 函数可以return函数（这个常在闭包中使用）
  • 函数有作用域
  • 函数可以作为构造函数
  • 函数其实是通过Function对象new出来的
  • 函数和对象的关系
  • ……

这些内容会在后面加以详细研究。
