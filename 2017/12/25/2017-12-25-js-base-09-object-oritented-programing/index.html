1、理解对象

创建对象最简单的方式是建立Object实例：

var person=new Object();
preson.name="Anna";
person.age=26;
person.job="Physician";
person.sayName=function(){
  alert(this.name);
}

或者是使用字面量法：

var person={
  name="Anna",
  age=26,
  job="Physician",
  sayName=function(){
      alert(this.name);
  }
};

对象吧，包含属性和方法。从实质上来说，对象只有属性，方法也是属性。因为函数名就是一个指针，调用函数要用函数名加括号。在对象里面，虽然调用属性不需要加括号，比如：person.age，但方法里面有函数，要调用属性的函数，但是不加括号，就只是把指针指向函数，比如：person.sayName；但要函数执行起来就要加括号，比如：person.sayName()。

在《JS高程》（第三版）P138页里面讲到：

ECMA-262把对象定义为：“对象是无序属性的集合，其属性可包含基本值、对象或者函数。”严格来讲，这就相当于说对象是一组没有特定顺序的值。对象的每个属性或方法都有一个名字，而每个名字都映射到一个值。正因为这样（以及其他将要讨论的原因），我们可以把ECMAScript的对象想象成散列表：无非就是一组名值对，其中值可以是数据或函数。

1、工厂模式

function person(name,age,job){
  //原料
  var o=new Object();
  //加工
  o.name=name;
  o.age=age;
  o.job=job;
  o.sayName=function(){alert(this.name);};
  //出厂
  return o;
}

var person1=person("Anna",26,"Doctor");
var person2=person("Bob",30,"writer");

2、构造函数模式

function Person(name,age,job){
  this.name=name;
  this.age=age;
  this.job=job;
  this.sayName=function(){alert(this.name);};//this.sayName=new Function(){"alert(this.name)"};
}

var person1=new Person("Anna",26,"teacher");
var person2=new Person("Bob",26,"teacher");

3、原型模式

function Person(){};
Person.prototype.name="Anna";
Person.prototype.age=29;
Person.prototype.sayName=function(){alert(this.name);};

var person1=new Person();

var person2=new Person();
    person2.name="Bob";

person1.sayName();
person2.sayName();
  • 原型属性
  • 实例属性：实例属性可以屏蔽同名原型属性

检测属性：

  • hasOwnProerty()：检测属性是否为实例属性。true/false
  • in：检测实例对象能否访问该属性。（无论是实例属性还是原型属性）。true/false
  • Object.keys()：返回对象可枚举的实例属性。（是实例对象自身的属性哦！）
  • Object.getOwnPropertyNames():返回所有实例属性，无论是否可枚举。

原型链

function Father(){
  this.Fcolor="red";
}
Father.prototype.getFatherColor=function(){
  alert(this.Fcolor);
}

function Son(){
  this.Scolor="blue";
}

//继承Father
Son.prototype=new Father();
Son.prototype.getSonColor=function(){
  alert(this.Scolor);
}

var instance=new Son();
instance.getFatherColor();
